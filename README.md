# 🐕 Breed-o-Pedia

A comprehensive dog breed encyclopedia that lets you explore various dog breeds, view image galleries, discover sub-breeds, and vote on your favorite dog photos using the Dog CEO API.

## 🚀 Features

- **Breed Selection**: Browse and select from all dog breeds
- **Sub-breed Support**: Automatically shows sub-breeds when available
- **Image Gallery**: View dog images in a responsive grid layout
- **Load More**: Pagination functionality to load additional images
- **Random Dog Generator**: Get random dog images with one click
- **Vote System**: Hot 🔥 / Not ❄️ voting system with localStorage persistence
- **Loading States**: Visual feedback during API calls
- **Error Handling**: Graceful error messages when API fails
- **Fully Responsive**: Optimized for mobile, tablet, and desktop devices

## 🛠️ Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Flexbox, CSS Grid, Media Queries
- **Vanilla JavaScript** - ES6+ features, Fetch API
- **REST API** - Dog CEO API (https://dog.ceo/dog-api/)
- **LocalStorage** - Persistent voting system
